# React Learning Projects

A collection of React projects covering core concepts from setup to form handling.

---

## 📁 Projects Overview

### 1. `SetUpReact`
> *Created: 02-Jun-26*

Getting started with React — environment setup, project scaffolding, and understanding the basic file structure of a React app.

**Topics Covered:**
- Installing Node.js and npm
- Creating a React app with `create-react-app` or Vite
- Understanding `public/` and `src/` directories
- Running the development server

---

### 2. `my-counter-app`
> *Created: 02-Jun-26*

A classic beginner React project — a counter application to learn state management basics.

**Topics Covered:**
- `useState` hook
- Event handling (`onClick`)
- Incrementing, decrementing, and resetting state
- Conditional rendering

---

### 3. `InputHandlingInReact`
> *Created: 04-Jun-26*

Exploring how React handles user input through controlled components.

**Topics Covered:**
- Controlled vs uncontrolled inputs
- `onChange` event handler
- Two-way data binding with `useState`
- Displaying live input values

---

### 4. `Form_Handling_In_React_01`
> *Created: 08-Jun-26*

Building and managing forms in React with validation and submission handling.

**Topics Covered:**
- Handling multiple form fields with state
- `onSubmit` event and preventing default behavior
- Basic form validation
- Resetting form after submission

---

## 🚀 Getting Started

Each project is self-contained. To run any project:

```bash
cd <project-folder>
npm install
npm start
```

---

## 🛠 Prerequisites

- [Node.js](https://nodejs.org/) (v16 or above)
- npm or yarn
- A code editor (e.g., [VS Code](https://code.visualstudio.com/))

---

## 📚 Learning Path

```
SetUpReact → my-counter-app → InputHandlingInReact → Form_Handling_In_React_01
```
